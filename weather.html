WeatherApp

Weather App

Dec 2019

A simple weather app that shows weather conditions of user inputted cities. App was created with HTML, CSS, JavaScript, and Node.js. The app pulls API data from google places from cities around the world and the DarkSky API that gives weather details of a given city.

Screenshots

Learning Outcomes

Due to not learning about API calls and not adding APIs to my school final project, I wanted to learn how to make API calls using javascript as I knew that I have to know this as a junior web developer. I built this app by referring to a tutorial. It taught me step by step how to properly make an API call and how to hide API keys as some APIs require payment after a certain amount of calls.

Problems Faced

I had some minor problems of creating my node server as well as connecting the API properly to my script.

Solution to Problems

I solved my server problems by just making sure all my syntax were correct. I had some syntax errors that caused the server to crash. I also had to reread the documentation of the DarkSky API and the google API to understand why my APIs were not connected properly.
