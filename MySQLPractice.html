MySQL Practice

Post Section

Nov 2019

A small assignment I completed for our Web Development class. This section allows users to make posts, delete posts, and like or unlike votes. The application was created with PHP and data was stored and updated in a MySQL database

Screenshots

Learning Outcomes

Understand how to handle GET and POST requests while writing php functions.

Create and connect a MySQL database.

Write and implement SQL queries to php files to pull data from MySQL database.

Problems Faced

Since the app had multiple php files, I was confused at first as to where I need to take or pass values from. As a result, there were many occasions where I got errors in the console that said variables were undefined.

Solution to Problems

Once I got a better understanding of the sequence of how I should be handling the data, it became clearer to me which php files I should be working on. Each ajax request should link to the right php file that executes a certain task. I then needed to check if it's a GET or POST request in the specified php file. Each php file is connected to another data.php file which contains all the SQL queries that I should be adding.
