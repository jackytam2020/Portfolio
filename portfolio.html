Portfolio

Portfolio Website

Dec 2019

Made with HTML, CSS, JavaScript, and Materialize CSS. The website uses various JavaScript modules from materialize to showcase my work and add responsiveness to the website.

Learning Outcomes

I learned how to use the framework library of Materialize CSS. Since I never learned how to use a framework css library in my program, I felt that I was necessary that I learned this on my own. This made styling extremely fast compared to only using bare css. I felt that this was a necessary skill in any junior web developer position..

Problems Faced

Working with the positioning of elements was an issue as well as knowing where to add my own custom css.

Solution to Problems

Simply reading the documentation of Materialize Css helped me understand exactly how the framework works. Some items from the library needs to be initialized first in order for it to work.
